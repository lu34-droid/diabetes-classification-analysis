# diabetes-classification-analysis

Overview

This project predicts diabetes using machine learning based on medical data such as glucose, BMI, and age.

Dataset
768 samples, 9 features
Target: Outcome (0 = no diabetes, 1 = diabetes)
Imbalanced data (500 vs 268)
Models
Logistic Regression (baseline)
Random Forest (improved)
Results
Logistic Regression: low recall (misses many cases)
Random Forest: better performance
Accuracy: 0.76
Recall: 0.74
Conclusion

Random Forest performs better, especially for detecting diabetes cases.
